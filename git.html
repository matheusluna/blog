Conhecendo o Git

     O Git é um sistema de controle de versão, onde podemos desenvolver projetos de forma colaborativa, ou seja, várias pessoas podem contribuir simultaneamente, editando arquivos já existentes ou criando novos arquivos no determinado projeto sem riscos de substituição do trabalho já existente.

     A importância do sistema de controle de versão se dá devido a organização que ele proporciona quando várias pessoas trabalham em conjunto no projeto, sem um sistema assim, projetos levariam muito mais tempo para serem finalizados, imaginando situações ondem dois ou mais colaboradores trabalham em um mesmo arquivo ao mesmo tempo, provavelmente uma boa parte da informação correria o risco de ser sobrescrita.

     Atualmente existem alguns serviçoes na web que oferecem diversas funcionalidades extras aplicadas ao Git, dentre os mais conhecidos, temos o GitHub, GitLab e Bitbucket. Eles oferecem um repositório remoto, onde você pode salvar o seu projeto e trabalhar nele localmente sem correr risco de perder suas informações acerca do projeto.

Instalando o Git

Linux:
Para distribuições Debian, como o Ubuntu, basta digitar no terminal: 
$ sudo apt-get install git
Ou para distribuições fedora, podemos usar o yum: 
$ yum install git-core
Mac:
Podemos instalar o Git no Mac através de duas maneiras, a primeira é utilizar o instalador gráfico disponível no SourceForge:
http://sourceforge.net/projects/git-osx-installer/
Ou podemos instalar via MacPorts, utilizando o comando: 
$ sudo port install git-core +svn +doc +bash_completion +gitweb
Windows:
Instalar o Git no Windows é muito fácil, simpliesmente baixamos o executável do link abaixo e logo em seguida instalamos:
http://msysgit.github.com


Para aprender mais sobre o Git, a documentação se encontra no link abaixo: 
https://git-scm.com/documentation
