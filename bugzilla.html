Tratando bugs: Bugzilla

Publicado por David no dia 24/09/2016 as 18:36 horas.

     O Bugzilla é uma ferramenta web de Gerenciamento de mudanças open source, responsável por gerenciar bugs reportados em projetos de desenvolvimento. Originalmente desenvolvido pela Fundação Mozilla para apoiar o desenvolvimento do seu navegado, o Mozilla Firefox, é atualmente utilizado em grande número de projetos de software tanto livres quanto proprietários. Pode ser utilizado em vários sistemas operacionais como Windows, Mac e Linux.

     As operações principais oferecidas ao usuário são simples e diretamente ligadas às atividades de manutenção: a criação de um novo bug; sua triagem, revisão, escalonamento, discussão e fechamento; e a obtenção de relatórios e consultas baseados em propriedades do bug. Para suportar essas operações, o Bugzilla implementa várias características, como:

  • Contas de usuário;
  • Formulário do Bug;
  • Comentários;
  • Gerenciador de anexos;
  • Interface de consulta;
  • Integração com o correio eletrônico.

Instalando o Bugzilla

Linux, Windows e Mac:
A melhor maneira de se obter o Bugzilla no linux (e Windows) é clonando através do Git (baixar o git para o Windows no site http://git-scm.com/download): 
git clone --branch release-X.X-stable https://github.com/bugzilla/bugzilla
Execute o comando acima na sua pasta Raiz (para o Windows use a pasta C:\bugzilla), substituindo o "X.X" pelo número de 2 digitos da versão que você deseja (ex: 4.4).

     A documentação do Bugzilla está disponível no seguinte link: 
https://www.bugzilla.org/docs/
