## Sinopse

Café com gerência é um blog desenvolvido para a terceira nota da cadeira de Gerência e configuração de mudanças do curso de Análise e Desenvolvimento de Sistemas do IFPB.

## Instalação

Basta iniciar o index.html usando um navegador qualquer ou abrir diretamente através da página hospedada no GitHub: matheusluna.githb.io/blog/

## Ferramenta(s) de desenvolvimento

* Bootstrapper -> desenvolvimento do layout
* Geany e Atom -> edição dos arquivos html e inserção das publicações
