Trac

Publicado e editado por Matheus no dia 27/09/2016 ás 11:23 hrs

O Trac é basicamente uma ferramenta open souce de interface web para gerenciamento de atividade e mudanças ocorridas durante o desenvolvimento do software. Essa ferramenta tem como recurso:

Controle de atividades;
Wiki para documentação;
Integração entre o controle de versão e o controle de mudanças;
Acompanhamento da evolução do projeto;

Instalando o Trac

Para baixar o trac basta clonar o repositório no github e começar a usar: 

$ git clone https://github.com/edgewall/trac.git

Link do site da ferramenta:

https://trac.edgewall.org/
